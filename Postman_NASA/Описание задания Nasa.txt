Тестовое задание выполняется на площадке open api NASA - https://api.nasa.gov/ 
Задание выполнять в Postman 
Задача:  
•	Необходимо найти запросы Mars Rover Photos 
•	Выполнить запрос по Querying by Earth date на дату 21.01.2022 
•	Передать в переменную окружения id второй фотографии, распарсив json
В данном задании я согласно документации создавала в Postman коллекцию и окружение NASA.
Научилась сохранять с данными API Key во вкладке Params. Выполнила GET  запрос, который дал ответ с фотографиями по заданному параметру: наименование марсохода и количеству марсианский сол.  
Выполнила GET  запрос по параметру Querying by Earth date на дату 21.01.2022. 
Научилась использовать сниппеты: для того, чтобы, распарсив json, вытащить из тела ответа от сервера id второй фотографии;  для установки переменной id второй фотографии в окружение; для просмотра в консоли логи по данному запросу.
