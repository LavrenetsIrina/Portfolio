Задание: Практика работы с REST API через GoRest
Цель: научиться отправлять запросы к REST API (GET, POST, PUT, DELETE), использовать токен авторизации, работать с вложенными ресурсами.
В данном задании я создала коллекцию запросов. Научилась использовать токен для авторизации через заголовок запроса. Научилась создавать окружение, и переносить коллекцию в данное окружение. Проводила анализ тела ответа от сервера с целью получения информации необходимой для заполнения в теле запроса. 
Объявляла переменную Base URL, значение которой сохраняла в окружении. Также объявляла переменными идентификаторы пользователя, поста и комментариев, значение которых сохраняла в коллекции. 
Научилась использовать сниппеты для установки переменных в коллекцию и в окружение.
